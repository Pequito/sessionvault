# Contributing to SessionVault

- Add tests
- Create plugins (see plugin API in README)
- Translate / add themes

## Pull Requests

- Target `main` branch
- Use descriptive titles
- Include tests if adding features/fixes
- Follow PEP 8 style
- Update CHANGELOG.md if user-visible change
